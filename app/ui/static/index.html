欢迎使用万能问答框架

万能问答框架是一个基于 RAG 技术的智能问答系统，支持用户上传知识库并生成专属的问答助手。

功能特点

  • 支持多种文件格式（Markdown、PDF、Word、TXT、JSON）
  • 自动处理知识库并生成向量存储
  • 一键创建专属问答助手
  • 支持 ReAct 推理流程
  • 提供完整的 API 接口

快速开始

  1. 在 知识库管理 页面上传您的知识库
  2. 等待知识库处理完成
  3. 在 助手管理 页面创建您的专属助手
  4. 在 聊天 页面与助手对话
